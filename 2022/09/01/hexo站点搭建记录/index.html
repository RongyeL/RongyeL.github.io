Hexo博客初步搭建

发表于 2022-09-01 更新于 2022-09-03
记录自己搭建hexo的过程。全是坑。

参考文章：

https://zhuanlan.zhihu.com/p/26625249\

https://zhuanlan.zhihu.com/p/60578464

搭建计划：

  • 设定主题为next
  • 版权声明
  • 导航栏配置
  • 侧边栏头像
  • 侧边栏社交小图标-b站未完成
  • 显示阅读进度
  • 增加搜索功能
  • 修改文章底部标签图标
  • 文章末尾增加本文结束
  • 网站底部增加访问量
  • 增加字数统计
  • 设置网站的图标
  • 修改网页底部的桃心
  • 博文置顶配置

解决方案

将hexo根目录下的.deploy_git/删除掉，然后使用hexo deploy重新部署就可以了。
但是暂时不清楚是什么原因引起的错误。

可以hexo d但是在打开网站后只能看到readme文件。

原因在于没有设置默认分支为master。自动建立了一个main branch。

在设置中设置默认分支，并删除掉main branch。

删除.deploy_git，并重新hexo d

现在可以在https://rongyel.github.io/

看到自己的博客界面啦。

更换为next主题后，本地服务可以看到页面已经换了

但是.io的页面还是原来的样子。

npm install hexo-renderer-pug hexo-renderer-stylus –save

安装以后再hexo d

不知道为什么就可以了。

每次hexo d推送之前，最好先清除浏览器的缓存。不然会发现没有变化。
